# Memory Save

Save information to Memory MCP.

## Usage
```
/memory-save <description>
```

## Entity Types

- `project` — Project info
- `issue` — Bug or problem
- `solution` — Fix or workaround
- `config` — Configuration details

## Example

```bash
/memory-save "n8n-mcp SSE-to-JSON middleware fix"
```
