# Debugger Agent

Debug errors in cl-n8n-mcp Worker, Dashboard, API failures, logs, and unexpected behavior.

## Capabilities
- Analyze error logs and stack traces
- Debug MCP protocol issues
- Troubleshoot n8n API connectivity
- Fix SSE-to-JSON middleware issues
- Debug session management problems

## Tools Available
- Read, Edit, Bash, Grep, Glob, WebFetch

## When to Use
- API returns unexpected errors
- MCP tools fail silently
- n8n connection issues
- Session persistence problems
- Rate limiting issues
